Atribut / Directive pada Alpinejs

x-data

Berfungsi sebagai inisiasi komponen Alpine, Selain itu mendeklarasi data pada komponen

Berfungsi untuk menjalankan kode JavaScript pada saat komponen pertama kali diinitialisasi

Contoh untuk x-init

x-show

Berfungsi untuk menampilkan element HTML jija nilai adalah true

Ini adalah contoh x-show
